# Fotos del fondo del login

Coloca aquí las fotos que quieres que aparezcan de fondo en la pantalla de
inicio de sesión. Se muestran una por una, con transición suave, en orden
alfabético de nombre de archivo.

- Formatos aceptados: `.jpg`, `.jpeg`, `.jfif`, `.png`, `.webp`, `.avif`, `.gif`
- Recomendado: fotos horizontales de al menos 1600px de ancho
- Sugerencia de nombres para controlar el orden: `01-plaza.jpg`, `02-iglesia.jpg`, …

Si esta carpeta no tiene imágenes, el login muestra el fondo verde de siempre.
